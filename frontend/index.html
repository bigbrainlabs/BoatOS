ℹ️ AIS-Integration

Was ist AIS?
AIS (Automatic Identification System) zeigt Positionen und Details anderer Schiffe in Echtzeit auf der Karte.

Funktionen:
• Schiffspositionen als Icons auf der Karte
• Schiffsdetails (Name, MMSI, Kurs, Geschwindigkeit)
• Navigationsstatus (unterwegs, vor Anker, etc.)
• Auto-Update alle 60 Sekunden

🔵 AISStream: Kostenlos, WebSocket-basiert, weltweite Abdeckung
🟡 AISHub: Benötigt API-Key (Registrierung erforderlich)

🚀 OSRM Routing (Empfohlen)

Was ist OSRM?
Open Source Routing Machine - Lokaler Routing-Server auf dem Raspberry Pi, der speziell für Wasserwege konfiguriert ist.

Vorteile:
⚡ Blitzschnell (< 100ms)
🌊 Folgt echten Wasserwegen (Flüsse, Kanäle, Fairways)
📡 Komplett offline
♾️ Unbegrenzte Anfragen
🆓 Kostenlos

Installation:
OSRM kann auf dem Raspberry Pi installiert werden. Nach der Installation wird es automatisch beim Booten gestartet.

☁️ GraphHopper Routing

Was ist GraphHopper?
Cloud-basierter Routing-Service über API.

Hinweis:
⚠️ Kostenlose Version routet über Land, nicht über Wasserwege
📶 Benötigt Internetverbindung
🔢 Begrenzt auf 500 Anfragen/Tag

Berücksichtigt Fließgeschwindigkeiten von Flüssen und Gewässern für genauere ETA-Berechnungen

ℹ️ Wie funktioniert das?

Strömungsberücksichtigung:
Die Fließgeschwindigkeit wird bei der ETA-Berechnung berücksichtigt:

• Stromabwärts: Effektive Geschwindigkeit = Bootsgeschwindigkeit + Strömung
• Stromaufwärts: Effektive Geschwindigkeit = Bootsgeschwindigkeit - Strömung
• Querströmung: Teilweise Berücksichtigung basierend auf Winkel

Datenquellen:
1. Live-Daten von PEGELONLINE (falls verfügbar)
2. Statische Werte aus dieser Tabelle (Fallback)
